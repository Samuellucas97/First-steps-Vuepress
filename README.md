It is only simple personal website developed with Vuepress...

# Informations


## Development

```bash
# install VuePress locally
npm i -g vuepress

# Start local dev server
vuepress dev portfolio
```

## Build

```bash
# Build static files to .vuepress/dist
vuepress build portfolio
```
